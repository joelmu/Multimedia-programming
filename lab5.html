1. What you can do with the jQuery?

With jQuery you can easily select and manipulate DOM elements, handle events and effects, use Ajax and JSON and you can use backwards compatiblity methods. Also, jQuery is possible to extend by plug-ins and it supports multiple browsers.

JQuery makes web developers work easier. With less writing you can do more.
