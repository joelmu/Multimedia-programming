1. What you can do with the jQuery?

With jQuery you can easily select and manipulate DOM elements, handle events and effects, use Ajax and JSON and you can use backwards compatiblity methods. Also, jQuery is possible to extend by plug-ins and it supports multiple browsers.

2. How it can help you as web developer?

JQuery makes web developers work easier. With less writing you can do more.

4. What are CDNs? What kind of benefits you get if you use them?

Content Delivery Network is a system of distributed servers that deliver WEB content to user, based on the user's geographic location, the origin of the webpage and the server that delivers the content.

Benefits of using CDN are:
  • Files may be pre-cached
  • Big CDN hosts have high-capacity
  • Built-in version control
  • Usage analytics
  • Boosts performance
